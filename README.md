# Smart GPA Planner & Academic Management System

A C-based academic management system developed as a team project for the **Computer Language and Programming** course at **Tsinghua University**. The application helps students manage academic records, monitor graduation progress, and plan future academic performance through GPA calculation and analysis tools.

---

## Overview

Smart GPA Planner is a command-line application designed for undergraduate students following a four-year degree program. It allows users to manage course records, calculate cumulative and major GPAs, monitor graduation progress, and explore different academic planning scenarios within a single system.

The project was developed using the C programming language and applies core programming concepts including functions, arrays, structures, file handling, sorting algorithms, and data validation.

## Sample Functionalities

The system enables users to:

- Create and manage student profiles
- Add, edit, search, and organize course records
- Automatically calculate cumulative GPA and major GPA
- Generate academic transcripts grouped by semester
- Track completed and remaining graduation credits
- Monitor semester performance trends
- Predict the GPA required to achieve future academic goals
- Export academic records to CSV format
- Import transcript data from CSV files
